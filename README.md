🩸 Wearable Blood Flow Monitoring (PPG Simulation)
📌 Project Description

This project demonstrates a conceptual design of a wearable device for non-invasive blood flow monitoring using a PPG (Photoplethysmography) sensor.

It includes an interactive simulation that visualizes how blood flow affects the PPG signal waveform in real time.

🚀 Features
📊 Interactive PPG waveform simulation
🎛️ Adjustable pulse amplitude using slider
🧠 Conceptual block diagram of system
🎨 Modern UI design using HTML & CSS
⚡ Real-time canvas-based signal rendering
🧠 What is PPG?

Photoplethysmography (PPG) is an optical technique used to measure changes in blood volume using light absorption in skin tissue.
